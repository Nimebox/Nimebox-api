# 🏃 Running application

### What do you need?

- [Node.js](https://nodejs.org/en/download/)

### 🤘 Let's start

- Clone repo
    - Via https
        ```bash
        git clone https://github.com/nimebox/nimebox-api.git
        ```
   - Via ssh
        ```bash
        git clone git@github.com:nimebox/nimebox-api.git
        ```

- Now enter to app dir
    ```bash
    cd nimebox-api
    ```
- Install depedencies
    ```bash
    npm i
    ```
- Running application
    ```bash
    npm run dev
    ```
- Running application in production mode
    ```bash
    npm run build && npm start
    ```
- Running linter
    ```bash
    npm run lint
    ```
- Running tests
    ```bash
    npm test
    ```
